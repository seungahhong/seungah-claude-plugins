# GEO 9가지 방법론 (프린스턴 대학 연구 기반)

## 연구 개요

프린스턴 대학이 2023년 11월 발표한 생성형 검색엔진 최적화(GEO) 연구. AI 검색엔진이 콘텐츠를 인용하는 방식을 분석하고, 가시성을 높이는 9가지 방법을 검증했다.

## 9가지 방법론 상세

### 1. 출처 인용 (+40%)
- 주장마다 권위 있는 출처를 명시
- "~에 따르면", "~가 발표한 바에 의하면" 형식 사용
- 학술 논문, 공식 보고서, 업계 리더 인용

### 2. 통계 추가 (+37%)
- 구체적 수치와 데이터 포인트 포함
- "약 X%" → "정확히 37.2%" (구체적일수록 효과적)
- 최신 통계 우선 사용 (1년 이내)

### 3. 인용구 추가 (+30%)
- 업계 전문가, 연구자의 직접 인용
- 출처 명확히 기재 (이름, 직함, 소속)
- 핵심 주장을 뒷받침하는 위치에 배치

### 4. 권위적 어조 (+25%)
- 확신 있는 전문가 문체
- "~일 수도 있다" → "~이다" (단정적 표현)
- 도메인 전문성이 드러나는 서술

### 5. 이해하기 쉽게 (+20%)
- 복잡한 개념을 일상 언어로 풀어쓰기
- 비유와 예시 활용
- 전문 용어에는 간략한 설명 병기

### 6. 전문 용어 (+18%)
- 도메인별 전문 용어 적절히 포함
- 검색 의도에 맞는 기술적 정확성
- 5번(이해하기 쉽게)과 균형 유지

### 7. 고유 어휘 (+15%)
- 동의어와 관련 표현 다양하게 사용
- 반복적 단어 사용 지양
- 의미적 풍부함 확보

### 8. 유창성 최적화 (+15-30%)
- 문장 간 자연스러운 연결
- 논리적 흐름 (서론→본론→결론)
- 짧은 문장과 긴 문장의 적절한 혼합

### 9. 키워드 남용 (-10%)
- **절대 금지** — AI 검색에서 가시성 감소
- 자연스러운 키워드 배치만 허용
- SEO를 위한 키워드 반복은 역효과

## 최적 조합

| 조합 | 효과 |
|------|------|
| 유창성 + 통계 | 최대 효과 |
| 출처 인용 + 권위적 어조 | 신뢰성 극대화 |
| 이해하기 쉽게 + 전문 용어 | 폭넓은 독자층 확보 |

## 도메인별 권장 전략

| 도메인 | 최우선 방법 |
|--------|-----------|
| 기술/IT | 전문 용어 + 통계 + 출처 인용 |
| 비즈니스 | 통계 + 권위적 어조 + 인용구 |
| 의료/건강 | 출처 인용 + 통계 + 이해하기 쉽게 |
| 법률 | 권위적 어조 + 출처 인용 + 전문 용어 |
| 교육 | 이해하기 쉽게 + 유창성 + 통계 |
| 전자상거래 | 통계 + 인용구 + 유창성 |
